## Usage

Import it in your MDX documents.

```mdx title="tickets.mdx"
import { API, APIInfo } from 'fumadocs-ui/components/api';

<API>
  <APIInfo method="POST" route="/tickets">
    ...
  </APIInfo>
</API>
```
